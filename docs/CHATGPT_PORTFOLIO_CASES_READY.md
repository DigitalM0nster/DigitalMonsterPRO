# DigitalMonster — подготовленные кейсы 02, 04, 05, 06, 07

Этот документ содержит подтверждённую информацию и готовый русский копирайт для пяти кейсов. Его можно целиком передать ChatGPT для генерации идей 3D-моделей и сцен.

## Общие правила

- Эталон структуры: кейсы 01 НИПИГАЗ и 03 MMK-1.
- Каждый кейс состоит ровно из пяти состояний `state_01` … `state_05` с якорями `0 / 0.25 / 0.5 / 0.75 / 1`.
- Не приписывать DigitalMonster стратегию, дизайн или разработку всего продукта, если подтверждена только вёрстка либо отдельный субподрядный модуль.
- Не выдумывать метрики, награды, конверсию и состав команды.
- Видео в новых сценах по умолчанию не использовать. Оно допустимо только по клику и только одно на проект, если позднее будет принято отдельное решение.
- Для проекта 04 запрещено использовать название, изображения, палитру и 3D-модель кроссовка от CRE+RETAIL. Это чужая устаревшая заглушка.
- Проектный accent необязателен: можно использовать общий акцент портфолио.
